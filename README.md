# Cats-and-Dogs-Classifier

A deep learning project that classifies images as cats or dogs using transfer learning (ResNet / VGG) in PyTorch.
Trained and tested on the Kaggle “Dogs vs Cats” dataset using Google Colab.

# Overview

This project demonstrates:

Image Augmentation (random crop, flip, rotation)

Transfer Learning with pretrained CNNs (ResNet18, VGG16)

Model fine-tuning & evaluation on custom data

# Results

Phase	Loss	Accuracy
Train	0.2012	92.58%
Val	0.1871	96.43%
